only for flex container: 

	flex-direction   -> row, row-reverse, column, column-reverse

	flex-wrap -> nowrap(default), wrap, wrap-reverse

	flex-flow -> combination of both direction and wrap

	justify-content -> center, flex-start, flex-end, space-around, space-between

	align-items -> center, flex-start, flex-end, flex-stretch

	align-content -space-between, space-around, stretch,  center

	--------------

Flex items:

	order -> numeric value to control the order of flex items

	flex-grow -> numeric value to render the dimentions according to the other flex items

	flex-shrink - To specify not to shrink elements dimentions

	flex-basis - width property of flex item, which will be initial/stating dimention

	flex - combination of flex-grow, flex-shrink, flex-basis -> flex: 2 0 200px;

	align-self - control the rendering direction of current element only.
